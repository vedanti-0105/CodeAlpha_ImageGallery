# 📸 Image Gallery Project (SK 1)

## 📌 Project Description
This project is a responsive and interactive Image Gallery built using HTML, CSS, and JavaScript.

It includes:
- Responsive grid layout
- Lightbox view
- Next and Previous navigation buttons
- Hover effects with smooth transitions
- Image filtering by category (Bonus)

---

## 🚀 Features

### ✅ Core Features
- Responsive image grid using CSS Grid
- Lightbox popup when clicking images
- Next / Previous navigation buttons
- Smooth CSS transitions and hover effects
- Fully responsive design (Mobile, Tablet, Desktop)

### 🌟 Bonus Features
- Image filters (Nature, Animals, Travel)
- CSS hover zoom effect
- Brightness effect on hover

## 💡 How It Works

### 1️⃣ Image Grid
Images are displayed using CSS Grid layout.
The layout automatically adjusts based on screen size.

### 2️⃣ Lightbox
- Clicking an image opens it in fullscreen.
- Dark background overlay appears.
- Smooth transition effect.

### 3️⃣ Navigation Buttons
- Previous and Next buttons allow image browsing.
- JavaScript updates the image dynamically.

### 4️⃣ Filter Buttons
- Users can filter images by category.
- JavaScript hides and shows images based on selected category.
